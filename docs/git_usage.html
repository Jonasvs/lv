Versionskontrolle mit Git

Installation von
Git Portable
für Windows
  • Source: https://git-scm.com/download/win
  • Download "64-bit Git for Windows Portable"
  • .exe Datei ausführen und Git im Verzeichnis "PortableGit" auspacken
  • Git-Shell über git-cmd.exe öffnen
Git-Shell Navigation 
mkdir data
cd data
cd ..
dir
exit
Repository klonen 
git clone https://github.com/openwebcc/lv lv
Hilfe anzeigen 
git help [command]
Status anzeigen 
git status
Datei(en) hinzufügen 
git add [file, dir]
Bei Verzeichnissen werden alle Dateien im Verzeichnis hinzugfügt
Änderungen anzeigen 
git diff
Änderungen committen 
git commit -m "Datei hinzugefügt" [file, dir, .]
Änderungen auf den
Server übertragen 
git push
Änderungen vom
Server abrufen 
git pull
Log der Commits anzeigen 
git log
